# Quick Design Spec: Settings

**Type**: New Small System
**Scope**: Minimal settings menu overlay. SFX volume slider, music volume slider. localStorage persistence. Accessible from the title screen and pause menu (if present). Pure UI ‚Äî no gameplay logic.
**Date**: 2026-04-09
**Estimated Implementation**: ~1.5 hours (S effort)
**Priority**: T2 (Ship If Time) ‚Äî first to cut if scheduling slips

---

## Overview

Settings is a tiny DOM overlay with two sliders and a back button. On slider change, it calls `audio.setSfxVolume()` / `audio.setMusicVolume()` and writes the value to localStorage. On load, it reads localStorage and applies the stored values to the Audio System.

Deliberately scoped to the minimum that makes the game feel polished. No key rebinding, no graphics settings, no language toggle ‚Äî those are post-jam.

---

## Core Rules

### 1. DOM structure

```html
<div id="settings" class="settings" style="display: none">
  <div class="settings-panel">
    <h1>SETTINGS</h1>
    <div class="setting">
      <label>SFX Volume</label>
      <input type="range" class="sfx-slider" min="0" max="100" value="80" />
      <span class="value-label">80</span>
    </div>
    <div class="setting">
      <label>Music Volume</label>
      <input type="range" class="music-slider" min="0" max="100" value="50" />
      <span class="value-label">50</span>
    </div>
    <button class="back-btn">Back</button>
  </div>
</div>
```

### 2. Load on init

On `createSettings`:
1. Read `localStorage['spaceRunner.sfxVolume']` (default 0.8)
2. Read `localStorage['spaceRunner.musicVolume']` (default 0.5)
3. Apply to Audio System via `setSfxVolume` / `setMusicVolume`
4. Update slider positions to match

### 3. Slider change handlers

On `input` event:
1. Value = slider.valueAsNumber / 100
2. Call `audio.setSfxVolume(value)` (or music)
3. Write to localStorage
4. Update value label text

### 4. Show / hide

```ts
settings.show();   // display: block
settings.hide();   // display: none
```

Called from the title screen's "Settings" button and the back button.

### 5. Dispose

- Removes overlay from DOM
- Removes all event listeners
- Idempotent

## Tuning Knobs

No tunable gameplay knobs. Default slider values come from Audio System defaults (`defaultSfxVolume: 0.8`, `defaultMusicVolume: 0.5`).

---

## Data Files

None. Settings lives purely in localStorage at runtime:
- `spaceRunner.sfxVolume` ‚Äî 0..1 float string
- `spaceRunner.musicVolume` ‚Äî 0..1 float string

### CSS

`public/styles/settings.css` ‚Äî static file.
